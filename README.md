# Repositório de Estudos: Testes Automatizados com Appium

Bem-vindo ao meu repositório de estudo, focado na automação de testes com Appium! Neste projeto, estou explorando e aprimorando minhas habilidades em automação de testes mobile, especificamente para realizar a automatização de uma simples operação de soma em uma calculadora.

## Objetivo do Projeto
O principal objetivo deste projeto é utilizar o Appium para automatizar a execução de uma soma na calculadora de um dispositivo móvel. A automação incluirá interações com a interface do usuário da calculadora e a validação do resultado usando o JUnit.
